Bem-vindo à **Core Network** — uma plataforma focada em scripts de alta qualidade para FiveM.

Aqui você encontra tudo o que precisa para instalar, configurar e aproveitar ao máximo nossos sistemas. Desde a configuração inicial até recursos avançados de API, nossa documentação cobre cada etapa do processo com clareza e objetividade.

## O que é a Core Network?

A Core Network desenvolve scripts para servidores de GTA Roleplay na plataforma **FiveM**, com foco em inovação, qualidade e desempenho. Nossos sistemas são criados para oferecer a melhor experiência tanto para os administradores quanto para os jogadores do servidor.
