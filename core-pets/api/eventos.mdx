---
title: "Eventos"
description: "Eventos do Core Pets disponíveis para integração com outros scripts."
---

O Core Pets expõe eventos para abrir e fechar a interface e para reagir a ações do sistema em outros scripts.

---

## `core-pets:openUI`

Abre o painel de pelúcias do jogador.

<CodeGroup>
  ```lua Client
  TriggerEvent("core-pets:openUI")
  ```

  ```lua Server
  TriggerClientEvent("core-pets:openUI", source)
  ```
</CodeGroup>

---

## `core-pets:closeUI`

Fecha o painel de pelúcias do jogador.

<CodeGroup>
  ```lua Client
  TriggerEvent("core-pets:closeUI")
  ```

  ```lua Server
  TriggerClientEvent("core-pets:closeUI", source)
  ```
</CodeGroup>

---

## `core-pets:onPanelOpen` *(read-only)*

Disparado automaticamente no client quando a interface do Core Pets é aberta. Use para reagir a esse evento em outros scripts.

```lua
AddEventHandler("core-pets:onPanelOpen", function()
    -- Exemplo: remove arma da mão ao abrir o painel
    SetCurrentPedWeapon(PlayerPedId(), `WEAPON_UNARMED`, true)
end)
```

<Note>
  Este evento é somente de leitura — não o dispare manualmente para abrir a interface. Use `core-pets:openUI` para isso.
</Note>

---

## Exemplo de integração

O exemplo abaixo mostra como integrar o Core Pets em um script de coleta de rota, aplicando bônus de valor e concedendo XP ao jogador.

```lua
-- server-side, em outro script
local corePets = exports["core-pets"]

RegisterNetEvent("minhaRota:concluir", function(rotaId)
    local src      = source
    local passport = Functions.getUserId(src)
    local baseValue = 5000

    -- Aplica bônus de CollectRoute
    local finalValue = corePets:Apply(passport, "CollectRoute", baseValue)

    -- Concede XP no pet com esse benefício
    corePets:GrantXP(passport, "CollectRoute")

    -- Paga o jogador
    giveMoney(passport, finalValue)
end)
```
